O : 광선의 시작점
D : 광선의 방향벡터
t : 임의의 거리

P(t) = O + Dt

C : 원의 중점
r : 반지름
||P(t) - C|| - r = 0
||O + Dt - C|| - r = 0
O - C : M
||Dt + M|| - r = 0
(Dt + M)*(Dt + M) - r * r = 0
Dt^2 + 2MDt + M^2 - r^2 = 0
A : D^2
B : 2MD
C : M^2 - r^2

A는 단위벡터이므로 크기의 제곱을 해도 1이므로 제외한다.
(-B +- rt B^2 - 4AC) / 2A
(-B +- rt B^2 - 4C) / 2
